# Currency Converter (JavaScript)

Web-based application to convert currencies in real-time using API data.

## Features
- Real-time currency conversion
- Fetch API and asynchronous JavaScript
- Dynamic DOM updates
- Event-driven user interface

## Technologies Used
- JavaScript, HTML, CSS
- DOM Manipulation, Async Programming, Fetch API

## How to Run
1. Open `index.html` in a browser
2. Enter amount and select currencies to convert
